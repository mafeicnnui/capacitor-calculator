# 📱 全屏显示和自定义图标更新

## ✅ 已添加的功能

### 1. 全屏显示优化
- **viewport-fit=cover** - 适配刘海屏和圆角屏幕
- **动态视口高度** - 使用 `100dvh` 适配不同设备
- **安全区域适配** - 自动处理状态栏和导航栏
- **状态栏配置** - 深色主题，匹配应用颜色

### 2. 自定义计算器图标
- **真实计算器外观** - 深色外壳，绿色 LED 显示屏
- **彩色按钮布局** - 数字、运算符、等号不同颜色
- **多尺寸支持** - 自动生成所有 Android 密度的图标
- **高质量渲染** - 使用 ImageMagick 生成清晰图标

## 🎨 图标设计特点

### 视觉元素
- **外壳**: 深灰色渐变 (#2c3e50 → #34495e)
- **显示屏**: 深色背景 + 绿色数字 (#00ff88)
- **数字按钮**: 灰色 (#95a5a6)
- **运算符**: 橙色 (#e67e22)
- **等号**: 绿色 (#27ae60)
- **清除**: 红色 (#e74c3c)

### 生成的图标尺寸
- **mdpi**: 48×48px
- **hdpi**: 72×72px  
- **xhdpi**: 96×96px
- **xxhdpi**: 144×144px
- **xxxhdpi**: 192×192px
